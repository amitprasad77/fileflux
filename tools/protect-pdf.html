Frequently Asked Questions

How secure is the password protection? +
FileFlux uses 128-bit AES encryption via the PDF specification standard. This provides strong protection for most use cases. For highly sensitive documents, consider additional security measures.
Can I remove the password later? +
Yes — use a PDF reader like Adobe Acrobat (File → Properties → Security) to remove the password if you know it. You can also use online PDF unlock tools.
What happens if I forget the password? +
Unfortunately passwords cannot be recovered. Always keep a backup of your original unprotected PDF and store the password securely in a password manager.
